# Flight Search

Flight Search is a kid-friendly fork of SearXNG, built with safety as the top priority.

## It includes:

- SafeSearch forced
- Kid-friendly search engines only
- A kid-safe AI
- An easy-to-switch-to interface
- Most importantly, 100% kid-safe search results
- Risk scoring 0-100

## How It Works

Flight Search uses a dynamic blocklists that are not allowed in searches.

If any of those words are found:

- All search results are blocked
- A message is shown to the user
- Depending on the search, a hotline may be shown as well
